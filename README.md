# Tatusoft Website

Site institucional da Tatusoft - Soluções em Software para PME

## Estrutura do Projeto

```
tatusoft-website/
├── index.html          # Página principal
├── css/
│   └── style.css      # Estilos principais
├── js/
│   └── main.js        # JavaScript principal
├── assets/            # Imagens e recursos
└── README.md          # Documentação
```

## Funcionalidades

- Design responsivo para todos os dispositivos
- Menu de navegação com hamburger menu para mobile
- Smooth scrolling entre seções
- Animações de entrada suaves
- Formulário de contato
- Header com efeito de scroll

## Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis CSS e Grid/Flexbox)
- JavaScript Vanilla
- Google Fonts (Inter)

## Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/mateus-pinheiro/tatusoft-website.git
```

2. Abra o arquivo `index.html` em um navegador web

## Desenvolvimento

Para fazer alterações no site:

1. Edite os arquivos HTML, CSS ou JavaScript conforme necessário
2. Teste as alterações no navegador
3. Commit e push para o repositório

## Estrutura de Seções

- **Header**: Navegação fixa com logo e menu
- **Hero**: Seção principal com chamada para ação
- **Sobre**: Informações sobre a empresa
- **Serviços**: Grid de serviços oferecidos
- **Soluções**: Soluções personalizadas
- **Contato**: Formulário de contato
- **Footer**: Rodapé com informações de copyright

## Personalização

Para personalizar o site:

1. **Cores**: Edite as variáveis CSS em `css/style.css`
2. **Conteúdo**: Modifique o texto diretamente no `index.html`
3. **Imagens**: Adicione imagens na pasta `assets/`
4. **Fontes**: Altere a fonte importada do Google Fonts
